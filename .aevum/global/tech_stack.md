# Technology Stack: Thoth

## Core
- **Language**: TypeScript (v5.x)
- **Engine**: Node.js (v18+)
- **Architecture**: Domain-Driven Design (DDD) & Event-Driven Architecture

## Extension Framework
- **Primary API**: VS Code Extension API
- **Communication**: Model Context Protocol (MCP) - Agentic Bridge
- **UI Components**: React (v18+) within Webview Panels

## Data & Persistence
- **State Management**: JSON Index (index.json)
- **Knowledge Base**: Markdown-based Living Memory (.md)
- **Sync Protocol**: GitHub API (via GitHub Mission Control)

## Frontend Styling
- **CSS**: Vanilla CSS with CSS Variables (Theme-aware)
- **Markdown Rendering**: Custom Markdown Parser with Mermaid Support
